# Map mural PoC test tiles

Procedural, feature-rich placeholder tiles for the illustrated-map mural proof of concept
(see `.claude/plans/illustrated-map-mural-poc.md`). **Not final art** — these are deliberately
busy, distinct test targets for validating CLIP identification separation and MindAR tracking.

Generated by `/tmp/gen_tiles.py` (PIL). To regenerate, re-run that script.

States: arizona, colorado, georgia, texas, wyoming. Colorado + Wyoming are the CO↔WY
"similar geometry, different art" stress-test pair.

Served at `https://popcode.app/assets/map-poc/<state>.png` once deployed — usable directly
as image URLs for `scripts/test-identify.mjs`.
